## ⚠️ Important: Data Migration Required

This version introduces a new data format. **Your existing data will not load automatically.**

### Before Installing:
1. Open your **current version** of ePanel
2. Go to **File → Export as JSON**
3. Save the backup file somewhere safe

### After Installing:
1. Open the **new version** of ePanel
2. Go to **File → Import**
3. Select your JSON backup file

Your folders, URLs, and notes will be restored.

## Installation
1. Download `ePanel.dmg` or `ePanel.zip`
2. For DMG: Open the disk image and drag ePanel to Applications
3. For ZIP: Extract and move ePanel.app to Applications

> **Note:** On first launch, you may need to right-click and select "Open" to bypass Gatekeeper since the app is not notarized.
